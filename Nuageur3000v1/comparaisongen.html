Observation

Ce nuage de mots illustre les personnages, emplacements et actions les plus importantes dans les 4 romans analysés.

Le mot avec le plus d'apparitions est "répondit", ce qui est caractéristique aux romans, à cause de tous les dialogues entre les personnages. (On déduit que "demanda" est présent selon la même cause).

On retrouve la plupart des personnages importants, comme "Mr. Fogg", "Capitaine Nemo", "Pencroff", etc... Jules Verne décrit beaucoup ses personnages ("seul", "homme", "pieds").

Les lieux où les histoires se déroulent sont, le plus souvent, la "mer" ou une "île", ainsi que le "Nautilus"; la description de l'environnement contient le plus souvent du "vent".

Le temps est décrit au long des récits : "temps", "heures", "jours". En plus, Jules Verne quantifie les éléments décrits avec "deux", "mille", "cent".

Enfin, le vocabulaire du mouvement est très présent dans les romans ("allait", "porte", "trouvait", "passage"...).
